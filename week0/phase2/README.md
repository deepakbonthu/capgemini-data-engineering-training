# Phase 2 – SQL to PySpark (Joins & Aggregations)

## Objective
To understand how to work with multiple datasets using joins and perform real-world data analysis using SQL and PySpark.

## What I Did
- Created customers and orders datasets
- Performed joins using customer_id
- Solved real-world queries like:
  - Total spending per customer
  - Top customers by spending
  - Customers with no orders
  - City-wise revenue
  - Average order amount
  - Customers with multiple orders

## Key Learnings
- SQL JOIN → PySpark join()
- GROUP BY → groupBy()
- SUM, AVG, COUNT functions
- Filtering using conditions
- Sorting using orderBy()

## Challenges Faced
- Understanding join types (especially left join / anti join)
- Writing aggregation queries correctly

## Outcome
Gained hands-on experience in combining datasets and solving real-world business problems using SQL and PySpark.
